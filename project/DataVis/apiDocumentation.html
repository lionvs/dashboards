Account
POST api/Account/Register

register new user

request body model:

var user = {
     UserName: “user”,
     Password: “pass”,
     ConfirmPassword: “pass”
}

POST api/Account/Logout

log out current user

POST /Token

login user

request body model:

var loginData = {
     grant_type: 'password',
     username: "login",
     password: password"
};

Dashboards (Require authorization)
POST /api/dashboard

Save dashboard in database

request body model:

var dashboard = {
     Title: title,
     Config: config,
     Description: description,
     DataSource: {
         Copy: copy,
         Original: original
     }
}

GET /api/dashboard/{id}

Returns dashboard by id

response : dashboard object

GET /api/dashboard

Returns all users dashboards

response : list of object

{
     Title : "title",
     Description: "desc",
     Id: "id"
}

DELETE /api/datashboard/{id}

delete dashboard from database

PUT /api/datashboard/{id}

edit dashboard by id

request body model:

var dashboard = {
     Title: title,
     Config: config,
     Description: description,
     DataSource: {
         Copy: copy,
         Original: original
     }
}
